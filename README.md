# Sales-and-Customer-Insights-Dashboard

# Overview
This project demonstrates how Power BI dashboards can be used to translate raw e-commerce transaction data into clear, decision-ready insights for sales growth, product strategy, and outlet performance.

Using Power BI, I analysed grocery retail data to uncover:

- Revenue drivers

- Product performance patterns
 
- Outlet=level sales differences

- Customer buying preferences

The dashboard is designed for business users, not analysts, enabling faster, data-backed decisions.

# Business Problem

E-commerce grocery platforms handle large volumes of transactional data but often struggle to convert it into actionable insights.

Key challenges addressed:

- Which products and categories drive the most revenue?

- How do outlet type, size, and location affect sales?

- What customer preferences influence buying decisions?

- How can sales trends guide inventory and expansion strategy?

# Strategic Recommendations

- Prioritize inventory and shelf visibility for high-performing categories

- Focus promotional efforts on dominant outlet types and locations

- Design outlet-specific strategies based on size and location performance

- Use sales trends to plan seasonal campaigns and stock replenishment

- Improve positioning of lower-performing product segments strategically (not uniformly)
